# Hi3Helper.Plugin.Endfield

[English](./README.md) · 简体中文

[Collapse启动器](https://collapselauncher.com/)的第三方插件，以支持 明日方舟：终末地 的下载/更新/启动

当前插件已可以正常使用（虽然一些功能暂时还没有测试，但是应该并不影响使用）

</div>

## 当前能够正常工作的功能

- [x] 检测是否为最新版本
- [x] 获取背景图、banner、新闻
- [x] 启动游戏/检测游戏
- [x] 下载游戏
- [ ] 社媒面板(当前暂不支持)

## 当前支持的服务器

- [x] 国服
- [x] 全球服

## 预计功能/未编写完成功能

- [ ] 更新（当前无1.0.0版本无法进行测试）
- [ ] 预下载（等待官方启动器预下载）
- [ ] 检查游戏完整性（启动器似乎暂时没有此功能的对应接口）
- [ ] 对接多个服务器（如b服）
