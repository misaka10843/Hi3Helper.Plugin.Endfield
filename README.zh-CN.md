## 当前能够正常工作的功能如下：
- [x] 检测是否为最新版本
- [x] 获取背景图、banner、新闻
- [x] 启动游戏/检测游戏
- [x] 下载游戏
- [ ] 社媒面板(查看插件对应数据结构中)

- [ ] 更新（当前无1.0.0版本无法进行测试）

- [ ] 预下载（等待官方启动器预下载）

- [ ] 对接多个服务器（如全球服，b服）
